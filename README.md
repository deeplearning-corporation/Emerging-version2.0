## 安装
下载适用于 Windows x64 的安装程序：可在此 GitHub 仓库点击 Releases → v2.0 → 下载 emerging-win32-installer.exe
下载 Emerging 安装后的内容（.zip 压缩包）：可以在此 GitHub 仓库点击 Releases →  2.0 → 下载 emerging-win32.zip

## 编译
需要在 Windows 编译
1.下载 Microsoft Visual C++ 编译器
2.下载 NASM 以编译 运行时库

编译 emerging.cpp 与 emerging-gui.cpp：
```batch
cl /EHsc /Fe:emerging.exe emerging.cpp
cl /EHsc /Fe:emerging-gui.exe emerging-gui.cpp
```
